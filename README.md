## Instalação

Clonando o repositorio do projeto

```bash
  git clone git@github.com:FullStack-lab/etapa01-bootcamp-Acauhi99.git
```

Instalanda dependencias do projeto

```bash
  cd frontend
  npm install
```

Rodar o Projeto

```bash
  npm run dev
```
